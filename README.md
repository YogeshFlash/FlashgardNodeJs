# FlashgardPro

This repository houses the source code for the FlashgardPro CRM and mobile application.

## Directory Structure

- **/backend/**: The backend API (REST/GraphQL) that powers both the CRM web interface and the mobile app.
- **/frontend/**: The web-based CRM frontend interface for administrators and internal users.
- **/mobile/**: The Flutter mobile application.
- **/docs/**: Project documentation, database schemas, and API specifications.

## Technology Stack (Proposed)

*   **Backend:** Node.js/Express, Python/Django, or Go (TBD)
*   **Frontend (CRM):** React, Next.js, or Vue.js (TBD)
*   **Mobile App:** Flutter
*   **Database:** PostgreSQL or MongoDB (TBD)
